# Dashboard Screenshots

This directory contains screenshots of all available dashboards for documentation purposes.

## Available Dashboards

### Cluster Level
- `cluster-resources.png` - Kubernetes Resource Usage (Cluster)
- `coredns-monitoring.png` - CoreDNS Monitoring
- `etcd-monitoring.png` - etcd Monitoring
- `apiserver-monitoring.png` - Kubernetes API Server
- `certificate-expiration.png` - Certificate Expiration Tracking

### Namespace Level
- `namespace-resources.png` - Kubernetes Resource Usage (Namespace)

### Pod Level
- `pod-resources.png` - Kubernetes Resource Usage (Pods)

## Adding Screenshots

To add screenshots:
1. Deploy the dashboards to your Grafana instance
2. Navigate to each dashboard
3. Take a screenshot (full page recommended)
4. Save with the appropriate filename in this directory
5. Update this README if adding new dashboards

## Screenshot Guidelines

- Use a resolution of at least 1920x1080
- Ensure the time range shows meaningful data
- Include the dashboard title and time range selector
- Use PNG format for better quality
